# Nebsoji PvP Trainer

A web-based app for Pokémon PvP matchup calculations.

## Setup

1. Ensure Node.js is installed (`node -v` and `npm -v` in terminal).
2. Open the project folder in Visual Studio 2022.
3. Run `npm install` in the terminal.
4. Run `npm start` to launch the app.
5. Open `http://localhost:3000` in your browser.

## Usage

- Enter up to three Pokémon for your team and the opponent's team, including IVs and level.
- Select shield counts (0, 1, or 2) for both teams.
- Click "Calculate Matchups" to see results, including best lead, switch recommendations, and movesets.
